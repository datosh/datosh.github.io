Postmortem: unRAID Flash Device Failure

Sun, Jan 23, 2022 3-minute read

Status: Complete, action items in progress.

Summary: unRAID’s OS Flash Device failed undeteced for 14 days, preventing server to successfully reboot, taking all internal services down, including pfSense VM which prevented home network from accessing internet.

Impact: 2 hours of limited internet access (mobile data plan). Internal services still offline, due to broken configuration.

Root Cause: Flash device failed undeteced. No backup & limited internet access delayed mitigation.

Trigger: Failing reboot of unRAID server.

Resolution: Since no backup of unRAID flash device was available, I reinstalled unRAID onto new flash device. Activation of unRAID trial license or requesting replacement key without internet access was impossible. Since unRAID does not support WiFi, direct usage of mobile data plan was not possible.
In the end, I used Windows Internet Connection Sharing (ICS) feature and mobile data plan to get wired home network temporarily back online, so unRAID key could be requested for new flash device. Assigned disks in unRAID array from memory to boot pfSense VM, and home network back online.
Original unRAID configuration was lost, and still needs to be re-created from memory to get all internal services back online.

Detection: Broken web UI, when trying to instantiate new VM. All other services were still running fine.

Action Items:

  • Restore unRAID configuration
  • Automated cloud backup of unRAID flash device
  • Automated backup of pfSense config
  • Document mobile data & windows router setup
  • Implement alerting of unRAID system log errors

Lessons Learned

What went well

  • Outage was fixed in a timely manner, despite many technical limitations of home network.
  • No critical internal service data was lost in process, only service configuration.

What went wrong

  • Should have saved more information (system log) or tried to save config (usb backup) before deciding to reboot faulty system.

Where we got lucky

  • Lucky to find Windows mobile data share feature.
  • Remembered disk assignments in unRAID arrays.
